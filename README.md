# 📂 Лабораторные работы по C++ (1 семестр)

Добро пожаловать в репозиторий с лабораторными работами за первый семестр! Здесь собраны практические задания, в ходе выполнения которых были полностью освоены базовые концепции, синтаксис и фундаментальные принципы программирования на языке C++, включая переход к объектно-ориентированному программированию (ООП).

---

## 🚀 Ключевые навыки и технологии
* **Язык программирования:** C++ (стандарты C++11 / C++17 / C++20)
* **Среда разработки:** MS Visual Studio
* **Инструменты:** Git, Работа с файловой системой, Отладка кода

---

## 📚 Содержание и изученные темы

Ниже представлен список лабораторных работ и ключевых тем, которые были успешно пройдены:

### 🔹 Блок 1: Основы синтаксиса и базовые конструкции
* **Лабораторная 1:** Знакомство со средой разработки, базовый ввод/вывод данных (`std::cin`, `std::cout`), линейные алгоритмы.
* **Лабораторная 2:** Условные операторы (`if`, `else if`, `else`), логические операции и ветвление логики программы.
* **Лабораторная 3:** Циклы (`for`, `while`, `do-while`), итерационные вычисления и вложенные циклы.

### 🔹 Блок 2: Работа с данными и структурами
* **Лабораторная 4:** Одномерные и многомерные статические массивы, алгоритмы сортировки (пузырьковая, выбором) и поиска.
* **Лабораторная 5:** Функции, передача аргументов по значению, ссылке и указателю. Область видимости переменных.
* **Лабораторная 6:** Работа с текстовыми и бинарными файлами (`fstream`, `ifstream`, `ofstream`), чтение и запись потоков данных.

### 🔹 Блок 3: Продвинутые темы и введение в ООП
* **Лабораторная 7:** Динамическая память (`new`, `delete`), динамические массивы, управление указателями и предотвращение утечек памяти.
* **Лабораторная 8:** Введение в структуры (`struct`) и классы (`class`). Инкапсуляция, конструкторы, деструкторы и основы объектно-ориентированного подхода.

## 🛠️ Как запустить проекты локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Откройте интересующую папку с лабораторной работой (например, `lab_4/ConsoleApplication1`) в вашей IDE (Visual Studio, CLion или VS Code).
3. Скомпилируйте и запустите главный `.cpp` файл.
